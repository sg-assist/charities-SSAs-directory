# Community Health Worker Knowledge Base

This directory contains simplified job aids and reference materials for **community health workers (CHWs)**.

## Vertical Code: `CHW`

## Content Principles

- Maximum 7th grade reading level
- No Latin medical terms — plain-language equivalents only
- Every answer about a patient ends with danger signs + referral prompt
- No medication dosing — refer to facility
